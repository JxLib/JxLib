Jx.Button.Multi Examples

API Reference: Jx.Button.Multi Extends Jx.Button

Basic Multi Button

Multi buttons are both a button and a menu that has a toggle set in it. Selecting an item from the list sets the button's action.

Toggled Multi Button

Multi buttons work like regular buttons and can be set to toggle as part of a button set.

Disabled Multi Button

Multi buttons can be enabled and disabled.
