Jx.Button.Color Examples

API Reference: Jx.Button.Color Extends Jx.Button See Also: Jx.ColorPalette

Color Buttons

The color button is an extension of the flyout style button. It has the same options as a regular button except that instead of an icon it has a color swatch.

The flyout contains a Jx.ColorPalette that is hooked up to the color swatch on the button.

Disabled Color Button

Color buttons can be enabled and disabled too.
